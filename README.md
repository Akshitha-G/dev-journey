# dev-journey

I'm Akshitha, based in Hyderabad 🇮🇳 — and this is where I document everything I learn as I teach myself to code.

Every day I study one topic, write down what I understood (and what confused me), and push it here. Every weekend I build something small with what I've been learning that week. No polished summaries, no certificate flexes — just honest, consistent progress.
